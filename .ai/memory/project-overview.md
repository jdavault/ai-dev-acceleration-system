# Project Overview

## What This Repo Is

This is the **AI Dev Acceleration System** — a productized consulting system that gets installed into engineering teams' repos to accelerate development velocity using AI workflows and agents.

## What We're Building

- A demo issue tracker app (`app/`) to prove the system works
- The installable AI system (`.ai/`) that IS the product

## What Success Looks Like

- A developer using this system ships features 2–5x faster
- PR quality improves through automated AI review
- Workflows are repeatable, consistent, and IDE-agnostic
- First paying client within 30 days
