# AI Dev + Agent Acceleration System

A practical, installable system for accelerating software development using AI workflows and agents.

This project demonstrates how to:

- 🚀 Increase developer velocity (2–5x)
- ⚙️ Standardize AI-assisted coding workflows
- 🤖 Introduce lightweight, high-impact agents (e.g., PR review)

---

# 🧠 Overview

Most teams already have access to AI tools.

What they lack is a **system**.

This repository shows how to implement a structured approach to:

- Prompt-driven development
- AI-assisted refactoring
- Automated code review via agents
- Repeatable workflows across a team

---

# 🎯 Goals

- Provide a **working reference implementation**
- Demonstrate **before vs after AI workflows**
- Validate that the system works:
  - Locally
  - In CLI environments
  - On VPS setups

---

# ⚙️ Core Components

## 1. CLAUDE.md (System Brain)

Defines how AI should behave inside the repository:

- Coding standards
- Project structure
- Prompt expectations
- Workflow conventions

---

## 2. Prompt Workflows

Standardized workflows for:

- Feature development
- Refactoring
- Bug fixing
- Code review

These ensure consistent, high-quality AI output.

---

## 3. CLI-Based Execution

Lightweight scripts and commands that:

- Run prompts
- Analyze diffs
- Generate structured outputs

Designed to be **IDE-agnostic** (works with VSCode, Cursor, or terminal).

---

## 4. PR Review Agent (MVP)

An AI-powered agent that:

- Reviews code changes
- Identifies issues
- Suggests improvements
- Acts like a senior engineer reviewing PRs

---

# 🧪 Experiment

This repo is used to validate the system through:

## Pass 1 — Without AI

- Manual implementation
- Baseline timing

## Pass 2 — With AI System

- Use workflows + CLAUDE.md + agent
- Measure time + quality improvements
